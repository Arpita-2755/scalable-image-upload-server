# Scalable Image Upload Server

## Objective
A backend system that:
- Uploads images to AWS S3
- Supports multiple server instances
- Uses NGINX as load balancer
- CI pipeline using GitHub Actions
- No database used

## Tech Stack
- Node.js (Express)
- AWS S3
- NGINX
- GitHub Actions
